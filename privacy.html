Política de privacidad

Última actualización: 31 de mayo de 2026

1. Finalidad

Esta web informa sobre el proyecto AVANCEM PER OLESA y sobre sus criterios de tratamiento responsable de información pública para la preparación supervisada de contenidos orgánicos.

2. Datos recogidos por esta web

Esta web estática no utiliza formularios, no crea cuentas de usuario y no almacena datos personales de visitantes. El proveedor de alojamiento puede aplicar sus propios registros técnicos y políticas.

3. Fuentes públicas y revisión editorial

Las herramientas internas del proyecto pueden revisar fuentes públicas previamente autorizadas para identificar temas de interés municipal. Los hallazgos requieren revisión humana antes de convertirse en ideas editoriales.

  • No se almacenan perfiles personales.
  • No se copian comentarios literales de vecinos.
  • No se almacenan datos sensibles ni se segmentan mensajes por atributos sensibles.
  • No se publican ni programan contenidos automáticamente.

4. Integraciones con plataformas

Cualquier integración técnica con plataformas externas debe limitarse a los permisos necesarios, someterse a revisión humana y cumplir las condiciones aplicables. Las credenciales no se publican en esta web.

5. Contacto

Para consultas sobre privacidad, escribe a seinfoges@gmail.com.
